# Bourse Tunisie Website

A static website about scholarship opportunities in Tunisia.  
The website presents scholarship conditions, university/major information, reviews, blog-style sections, partner logos, and a contact/registration form.

## Features

- Responsive homepage with image slider
- About section with video
- Scholarship conditions section
- Reviews slider
- Blog/news cards
- University logo slider
- Contact and registration form
- Footer with contact information and social links

## Technologies Used

- HTML
- CSS
- JavaScript
- SCSS
- Font Awesome
- Swiper.js
- LightGallery.js
- Google Fonts

## How to Run

Open `index.html` directly in your browser.

No installation is required because this is a static HTML/CSS/JavaScript website.

## Notes

The project uses external CDN links for icons, fonts, sliders, and gallery effects, so an internet connection may be needed for all visual features to load correctly.
